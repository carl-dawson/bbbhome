Résumé Home Automation

Hi, I'm Carl Dawson

I'm a physicist/computer engineer working at Stanford University.

This Flask powered website is hosted on a Beaglebone Black micro-computer making use of the LAMP stack. The Beaglebone is connected to some custom circuitry, resulting in a versatile, scalable home automation web application. Currently, the system functions as a thermostat with an on/off switch, live temperature display, and temperature set point.
